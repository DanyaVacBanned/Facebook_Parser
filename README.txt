Исполняемый файл - main.exe
Список слов лежит в файле words.txt
Список групп в файле groups.txt
В целях корректной работы программы добавлять новые записи в списки следует с новой строки.

В программе используется библиотека selenium, необходимая для обхода блокировки скраппинга на сайтах.
Для корректной работы следует установаить браузер Chrome последней версии
